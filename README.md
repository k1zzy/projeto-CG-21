# Projeto de Computação Gráfica

Este projeto é uma simulação simples de um carro e uma garagem em 3D. Podes conduzir o carro por aí, interagir com ele (abrir portas, ligar luzes) e estacionar na garagem que também abre.

## Teclas

**Carro:**
*   **W / S**: Acelerar e Travar (ou Marcha-atrás)
*   **A / D**: Virar (Esquerda / Direita)
*   **1**: Ligar/Desligar Faróis
*   **3 / 4**: Abrir/Fechar Portas da Frente (Direita e Esquerda)
*   **5 / 6**: Abrir/Fechar Portas de Trás (Direita e Esquerda)

**Garagem:**
*   **O**: Abrir/Fechar Portões da Garagem

**Câmara:**
*   **Rato (Movimento)**: Olhar à volta
*   **Rato (Scroll)**: Zoom in / Zoom out
*   **2**: Vista "Condutor" (1ª Pessoa) / Vista "Orbital" (Fora do carro)
*   **7 (ou C)**: Alternar para Modo Livre (voar pelo mapa)
    *   **No Modo Livre:** W/A/S/D para mover, Q/E para subir e descer.
